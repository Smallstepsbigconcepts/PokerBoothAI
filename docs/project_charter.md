# PokerBoothAI Project Charter

## Mission

Transform ClubWPT Streamlabs recordings into automated poker entertainment content.

The system should identify interesting moments, reconstruct key hand information, generate commentary, and eventually produce podcast and video content with minimal human intervention.

---

## Success Criteria

Input:

* 1 to 3 hour ClubWPT Streamlabs recording

Output:

* Interesting moments detected
* Hand summaries generated
* Commentary generated
* Podcast-ready script generated
* Video-ready content generated

---

## Phase 0 - Foundation

Status: Complete

Deliverables:

* GitHub repository
* Cloud storage
* Sample footage
* Layout validation

---

## Phase 1 - Moment Detection

Status: Active

Goal:

Convert a recording into a timestamped list of potentially interesting poker moments.

Examples:

* Large pots
* All-ins
* Showdowns
* Significant stack changes
* Long decision points

Deliverable:

Timestamped event list.

---

## Phase 2 - Hand Reconstruction

Goal:

Extract hand information from detected moments.

Target Data:

* Hero cards
* Board cards
* Pot size
* Stack sizes
* Seat positions
* Actions

Deliverable:

Structured hand summaries.

---

## Phase 3 - Poker Analysis

Goal:

Analyze reconstructed hands.

Deliverable:

Strategic commentary and hand evaluations.

---

## Phase 4 - Commentary Engine

Goal:

Generate entertaining discussions between AI commentators.

Initial Personas:

* Grinder Greg
* Gambler Bob
* Solver Sam

Deliverable:

Commentary scripts.

---

## Phase 5 - Podcast Generation

Goal:

Convert commentary into audio content.

Deliverable:

Podcast episodes.

---

## Phase 6 - Content Factory

Goal:

Automatically create multi-platform content.

Targets:

* YouTube
* Shorts
* TikTok
* Podcast feeds

Deliverable:

Published-ready content assets.

---

## Development Rule

Only one active milestone at a time.

Current Milestone:

Phase 1 - Moment Detection.
